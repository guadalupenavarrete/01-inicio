¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML

HTML (HyperText Markup Language o lenguaje de marcado de hipertexto) es el lenguaje de etiquetas que funciona como una de las piedras angulares de la World Wide Web.

HTML es un lenguaje de etiquetas porque los elementos se representan con codigos Ej etiquetas: < h1< h2&alt; h3&alt; h4&alt; h5&alt; h6&alt; < p< title</p>
